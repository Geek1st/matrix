# 变更日志

本文件记录 Matrix Match-3 Demo 的所有重要变更。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2026-06-11

### 新增
- **核心游戏功能**
  - 6x6 宝石棋盘，支持 6 种宝石类型
  - 触摸/鼠标交互，支持点击和滑动交换
  - 三连匹配检测算法（横向和纵向）
  - 级联消除系统，支持连续消除
  - 重力下落和新宝石生成
  
- **特殊宝石系统**
  - 四连消除生成条纹宝石（横向/纵向）
  - 五连消除生成彩虹宝石
  - 特殊宝石激活效果（整行/整列/同色消除）
  
- **动画系统**
  - 宝石交换动画（平滑过渡）
  - 下落动画（带弹跳效果）
  - 消除爆炸粒子效果
  - 分数弹出动画
  - 连击屏幕震动效果
  
- **音效系统**
  - Web Audio API 实现
  - 交换音效、消除音效、连击音效
  - 无效操作提示音
  - 静音切换功能
  
- **UI 和反馈**
  - HUD 显示（分数、步数、连击数）
  - 游戏结束界面
  - 提示按钮（显示可行交换）
  - 触摸涟漪反馈
  - 处理中状态提示
  
- **响应式布局**
  - 自适应屏幕尺寸
  - 移动端触摸优化（最小 44px）
  - 横竖屏支持
  - 高 DPI 屏幕适配
  
- **数据持久化**
  - localStorage 自动保存
  - 游戏状态恢复
  - 存档版本控制
  
- **无障碍功能**
  - 色盲模式（图案区分）
  - 屏幕阅读器支持
  - 键盘导航
  
- **文档**
  - 用户指南 (demo-README.md)
  - 浏览器兼容性测试清单 (BROWSER_TEST.md)
  - 移动端测试清单 (MOBILE_TEST.md)
  - 架构图 (ARCHITECTURE.md)
  - 开发计划 (DEVELOPMENT_PLAN.md)
  - 差距分析 (DEMO_GAPS.md)

### 技术实现
- 纯 HTML5 + Canvas 实现
- 模块化 JavaScript 架构
- 面向对象设计
- 异步动画处理
- 粒子系统
- 事件驱动架构

### 已知问题
- 无已知严重问题
- 低端设备可能帧率较低
- 部分浏览器可能不支持 Web Audio API

## 版本说明

### 版本号规则
- **主版本号**：不兼容的 API 变更
- **次版本号**：向下兼容的功能性新增
- **修订号**：向下兼容的问题修正

### 变更类型
- **新增**：新功能
- **变更**：现有功能的变更
- **弃用**：即将移除的功能
- **移除**：已移除的功能
- **修复**：问题修正
- **安全**：安全性修复
