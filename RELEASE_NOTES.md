# Matrix Match-3 Demo v1.0.0 发布说明

**发布日期**: 2026-06-11  
**版本**: 1.0.0  
**状态**: 正式发布

## 🎮 关于

Matrix Match-3 是一款经典的三消益智游戏，采用纯 HTML5 + Canvas 技术实现，无需任何框架依赖，可直接在浏览器中运行。

---

## ✨ 主要特性

### 核心玩法
- **6x6 宝石棋盘**：6 种不同颜色和形状的宝石
- **三连消除**：横向或纵向匹配 3 个或更多相同宝石
- **级联消除**：消除后自动触发连锁反应
- **步数限制**：30 步内获得最高分

### 特殊宝石
- **条纹宝石**（4 连消除）：激活时清除整行或整列
- **彩虹宝石**（5 连消除）：激活时清除所有同色宝石

### 视觉效果
- **流畅动画**：交换、下落、消除全程动画
- **粒子特效**：消除时的爆炸粒子效果
- **分数弹出**：实时显示获得的分数
- **连击震动**：3 连击以上时屏幕震动
- **触摸反馈**：点击时的涟漪效果

### 音效系统
- **Web Audio API**：纯代码生成音效，无需音频文件
- **动态音调**：连击时音调逐渐升高
- **静音控制**：一键切换静音模式

### 响应式设计
- **自适应布局**：支持各种屏幕尺寸
- **移动端优化**：触摸目标 ≥ 44px
- **横竖屏支持**：自动适配屏幕方向
- **高 DPI 支持**：Retina 屏幕清晰显示

### 数据持久化
- **自动保存**：实时保存游戏进度
- **断点续玩**：关闭浏览器后可恢复
- **版本控制**：存档格式向后兼容

### 无障碍功能
- **色盲模式**：图案区分，不仅依赖颜色
- **屏幕阅读器**：关键信息可访问
- **键盘导航**：支持键盘操作

---

## 🎯 游戏目标

在 30 步内获得尽可能高的分数。

### 计分规则
- 基础消除：宝石数 × 10 分
- 连击加成：第 N 次连击 × N 倍分数

### 策略提示
- 寻找能触发级联消除的交换
- 优先创造 4 连或 5 连获得特殊宝石
- 合理激活特殊宝石获得最大收益

## 📦 安装和使用

### 方式一：直接打开
1. 下载 `matrix-match3-demo-v1.0.0.html`
2. 双击用浏览器打开
3. 开始游戏

### 方式二：本地服务器
```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js
npx http-server -p 8000
```

然后访问 `http://localhost:8000/matrix-match3-demo-v1.0.0.html`

### 方式三：打包分发
```powershell
# 运行打包脚本
.\scripts\package-demo.ps1 -Version "1.0.0"
```

生成的文件在 `dist/` 目录中。

## 🐛 已知问题

### 性能
- 低端设备（2GB RAM）可能出现帧率下降
- 复杂级联消除时可能短暂卡顿

### 兼容性
- 部分旧版浏览器不支持 Web Audio API（音效降级）
- IE 浏览器不支持（使用现代 Web 标准）

### 存档
- 清除浏览器缓存会丢失存档
- 隐私模式下无法保存
